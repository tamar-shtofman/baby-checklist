🎀 הגדרת הרשימה

כדי שהרשימה תסתנכרן בין שניכם בזמן אמת, צריך Firebase (חינם, 5 דקות).

  1. כנסי ל-console.firebase.google.com עם Gmail
  2. לחצי Create a project ← תני שם ← המשיכי עד סוף
  3. Realtime Database ← Create Database ← Test mode ← Enable
  4. Project Overview ← Add app ← Web </> ← Register app
  5. תקבלי בלוק firebaseConfig — העתיקי את כל האובייקט ← הדביקי למטה

שלחי לינק לקובץ + Config גם למתן — הרשימה תסתנכרן.
